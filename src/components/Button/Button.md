#### **Examples** ####

The default button looks like this:

Default
```jsx
<Button label="Default" type="default" />
```

Ghost
```jsx
<Button label="Ghost" type="ghost" />
```

Default small
```jsx
<Button label="Default small" type="default" size="small" />
```

Ghost small
```jsx
<Button label="Ghost small" type="ghost" size="small" />
```
